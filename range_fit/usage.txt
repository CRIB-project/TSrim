- Start SRIM
- TRIM Stopping / Range Tables
- Set up Ion and Target
  -- Lowest Ion Energy = 1 (keV)
  -- highest Ion Energy = 1000000 (keV)
- Calculate Table
  -- Output file name: "SRIM Outputs\[Element] [Mass number] in [Material]"
     e.g. SRIM Outputs\Hydrogen 1 in Mylar

- Edit range_fit/get_fit_prm.pl
  - Z, A, matterial name, input file name, output file name
- Execute get_fit_prm.pl
  - $ cd range_fit
  - $ perl get_fit_prm.pl
- Then range_fit*.txt includes fitting parameters of log(range) vs log(E)
